የገንዘብ ምንዛሬ መቀየሪያ (Currency Converter App)
ይህ መተግበሪያ ተጠቃሚዎች የኢትዮጵያ ብርን ወደ አሜሪካ ዶላር እና በተቃራኒው እንዲቀይሩ ይረዳል። መተግበሪያው ትክክለኛውን የምንዛሬ ዋጋ ከኢንተርኔት (Live API) ያመጣል።

ዋና ዋና ባህሪያት (Features)
Live Rates: የቅርብ ጊዜውን የምንዛሬ ተመን ከ API ያመጣል።

Persistent History: የልውውጥ ታሪኮች በብሮውዘሩ LocalStorage ውስጥ ስለሚቀመጡ ገጹ ቢታደስ እንኳ አይጠፉም።

History Management: ተጠቃሚዎች የሰሩትን ታሪክ ማየትና እንደፈለጉ ማጽዳት ይችላሉ።

Responsive Design: በኮምፒውተርም ሆነ በሞባይል ስልክ ለመጠቀም ይመቻል።

የተጠቀሙባቸው ቴክኖሎጂዎች (Technologies Used)
HTML5 & CSS3: ለገጹ መዋቅርና ውበት።

JavaScript (ES6+): ለሂሳብ ስሌቶችና ለተግባራት።

Fetch API: የውጭ ዳታዎችን ለማግኘት።

LocalStorage: ዳታዎችን በብሮውዘር ለማስቀመጥ።

እንዴት መጠቀም ይቻላል? (How to use)
መቀየር የሚፈልጉትን የገንዘብ መጠን ያስገቡ።

የምንዛሬ አቅጣጫውን (ከብር ወደ ዶላር ወይም ከዶላር ወደ ብር) ይምረጡ።

"ቀይር" የሚለውን በተን ይጫኑ።

ውጤቱን በስክሪኑ ላይ ያያሉ፣ እንዲሁም ከታች ባለው "የልውውጥ ታሪክ" ዝርዝር ውስጥ ይገባል።
